# Jau Notes

[Original document location](https://jausoft.com/cgit/jaunotes.git/about/).

## Git Repository
This project's canonical repositories is hosted on [Gothel Software](https://jausoft.com/cgit/jaunotes.git/).

## Goals
Simple notekeeping of configurations and tools, mostly related to Linux and FreeBSD.

This is a project agnostic version of my earlier [JogAmp Scripting Notes](https://jausoft.com/cgit/jogamp/jogamp-scripting.git/).

## Some Jausoft Projects
- [jaulib](https://jausoft.com/cgit/jaulib.git/about/)
  - Provides general C++ collections, algorithms and utilities
- [Gamp](https://jausoft.com/cgit/gamp.git/about/)
  - Addresses hardware accelerated graphics, audio, multimedia and processing.
  - Supports WebAssembly (Wasm) targets to also run within the web-browser
- [gfxbox2](https://jausoft.com/cgit/cs_class/gfxbox2.git/about/)
  - This project provides a C++ sandbox for computer graphics for our computer science class.
  - Its examples demonstrate basic usage and physics simulations.
  - Supports WebAssembly (Wasm) targets to also run within the web-browser
- [Direct-BT](https://jausoft.com/cgit/direct_bt.git/about/)
  - Provides direct [Bluetooth LE and BREDR](https://www.bluetooth.com/specifications/bluetooth-core-specification/) programming,
    offering robust high-performance support for embedded & desktop with zero overhead via C++ and Java.
- [Cipherpack](https://jausoft.com/cgit/cipherpack.git/about/)
  - *Cipherpack*, a secure stream processor utilizing public-key signatures to
    authenticate the sender and public-key encryption of a symmetric-key for multiple receiver
    ensuring their privacy and high-performance message encryption.
